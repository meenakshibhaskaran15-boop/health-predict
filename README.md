# HealthPredict AI

HealthPredict AI is a modern web application that leverages Artificial Intelligence to predict potential health risks based on user symptoms, lifestyle habits, and basic medical data.

## Features

- **AI Prediction Engine**: Powered by a RandomForest machine learning model.
- **Modern UI**: Built with React and Vanilla CSS for a premium, glassmorphism-inspired aesthetic.
- **Health Assessment Form**: Comprehensive multi-step assessment of symptoms and lifestyle.
- **Results Dashboard**: Detailed risk breakdown with color-coded alerts (Low/Medium/High).
- **Report Generation**: Download a professional PDF report of the assessment.

## Tech Stack

- **Frontend**: React (Vite, TypeScript), Framer Motion, Lucide React, Recharts, jsPDF.
- **Backend**: Python, FastAPI, Scikit-learn, Pandas.
- **Development**: Fully responsive and mobile-friendly design.

## How to Run

### Backend
1. Navigate to the `backend` folder.
2. Ensure you have Python installed.
3. (Optional) Activate the virtual environment: `.\venv\Scripts\Activate.ps1`
4. Run the API: `python main.py`

### Frontend
1. Navigate to the `frontend` folder.
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
